# ZWPullMenuView

###框架详细使用介绍：[我的简书](http://www.jianshu.com/p/30cbe5a6c104)

[http://www.jianshu.com/p/30cbe5a6c104](http://www.jianshu.com/p/30cbe5a6c104)

下拉选择视图、支持微信、支付宝等样式。自动计算最优显示位置、动画效果。

- 支持支付宝首页、微信首页下拉Menu样式。
- 自定义Menu样式
- 自动优化视图方向
- 允许自定义下拉位置
- 支持纯文字、图标文字混合多种模板
- 丰富的初始化方法、智能判断显示模板
